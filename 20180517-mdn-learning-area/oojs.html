JavaScript Objects

What is kept in JS Objects?

Object Properties: data stored in an object

Object Methods: functions stored in an object

An object written out is called an object literal. This is different from an object instantiated from a class.

Dot notation

Accessing data in an object with a dot, as in:

  • person.age
  • person.interests
  • person.bio

Why is dot notation used? To access an object's namespace and retrieve the object properties and methods encapsulated in the object.

Bracket notation

Brackets can also be used to access object namespaces:

  • person['age']
  • person['name']['first']

Setting object members

One can use dot or bracket notation to access a field and update it. A new field can also be created and set by using a new name with dot or bracket notation.

Dot notation cannot accept a variable value that points to a name; it has to have a literal member name.

this.<field>

This refers to the object in which code is being written. This allows the same code snippet to be used inside different objects, with no needed change.

Using Objects in JavaScript:

The methods stored in an object are available in an instance.
